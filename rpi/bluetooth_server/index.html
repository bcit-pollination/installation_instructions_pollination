Bluetooth server installation instructions

Overview#

In this tutorial you will:

  • clone the Pollination Voting Station for Raspberri Pi (RPI) GitHub repository,
  • install required dependencies and packages, and
  • build and run the Pollination Voting Station application on your RPI with Electron, MongoDB.

Requirements#

To successfully complete this tutorial you will need:

  • a USB with enough space and a name without spaces (ex: "USB", not "U SB"),
  • a Raspberry Pi with:
    • Raspbian OS
    • Bluetooth
    • Node js version 8.9.4
    • Mongo version 2.4.14
    • Python 3
  • a touchscreen (optional).

Setup#

1. Installing MongoDB#

  1. Update your machine
sudo apt update
sudo apt upgrade
  1. Begin the installation
sudo apt install mongodb
  1. Check the installed MongoDB version
mongo --version
Note

Make sure the version is 2.4.14.

  1. Navigate to the root folder of your system.
cd /
  1. Create a data/db/ directory.
sudo mkdir data/db/
  1. Start MongoDB service.
sudo service mongodb start
  1. Confirm a mongod.lock file was created inside data/db/.
cd data/db
ls

  1. Check for MongoDB connections.
sudo mongod
  1. Enter the MongoDB shell and connect to the database.
mongo

2. Installing Node JS#

  1. To download and install newest version of Node.js, use the following command
curl -sL https://deb.nodesource.com/setup_8.x | sudo -E bash -
  1. Begin the installation
sudo apt-get install -y nodejs
  1. Check the installed Node.js version
node -v
Note

Make sure the version is 8.9.4.

3. Setting up your Bluetooth#

Enter the following commands to set up your Raspberry Pi's Bluetooth

sudo service bluetooth stop
sudo update-rc.d bluetooth remove
sudo systemctl stop bluetooth
sudo systemctl disable bluetooth
sudo apt-get install bluetooth bluez libbluetooth-dev libudev-dev
Important

At this stage, please restart your RPI

Important

Once restarted, make sure that your Bluetooth is turned on.

Getting Started#

1. Cloning the project#

  1. Choose a directory of your choosing and clone the project at https://github.com/bcit-pollination/voting_station_pollination.git onto your Raspbian
git clone https://github.com/bcit-pollination/voting_station_pollination.git
  1. Navigate to the root directory of the project
cd voting_station_pollination/
  1. Install the required dependencies
npm install
pip install cryptography
  1. Check your mongoose version.
npm list | grep "mongoose"
Note

Make sure the version is 2.9.10.

2. Running the Electron Application#

  1. From the root directory, navigate to the touchingscreen_electron folder
cd touchscreen_electron
  1. Run the program
npm start

Run Control Server#

  1. Click on "Run Control Server".

  2. Log in with the credentials and click "Login".

  3. After you are done logging in, click on ‚ÄúGet Election List‚Äù.

    The application will generate the current Election List that the logged in organization has.

  4. Click on "Download Election Package".

    The "Usb List" section will be displayed.

  5. Plug your USB into the Raspberry Pi.

  6. Click on "Refresh" to have your USB path showed in the application.

  7. Click on the USB path of your choosing to have the Election Data exported to.

  8. Click on "Export Election Data".

  9. At this moment, you will see the Encoded Election Data exported to the root of your USB's root directory

  10. Open another terminal and enter the MongoDB shell to switch to the Pollination database.

    mongo
    use pollination

  11. Go back to the Navigation Page.

Run Voting Server#

  1. Click on "Run Voting Server".

  2. Log in with the credentials and click "Login(as Voting Pi)".

  3. Enter a location and click "Submit".

  4. Click on "Refresh" to have your USB path showed in the application.

  5. Choose your USB with the exported Election Data saved on it.

  6. Click on "Import Election Data".

  7. Click "Start BLE Server" to start the voting server.

Now, users will be able to vote on the Raspberry Pi touchscreen or connect to the Raspberry Pi on their mobile devices via Bluetooth.
