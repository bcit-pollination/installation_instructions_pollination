MySQL Database installation instructions

Overview#

In this tutorial you will:

  1. Install and starting MySQL server 8.0
  2. Generate required MySQL tables and stored procedures
  3. Set up MySQL database user

Requirements#

  • Ubuntu/Debian based OS
  • Installed git
  • Port 3306 is open

Procedures#

Install MySQL server#

  1. Download and save the MySQL release package from the APT repository https://dev.mysql.com/downloads/repo/apt/

    note

    This package only works for Ubuntu/Debian based operating systems. You will need to look online to see how to install on other operating systems such as MacOS.

  2. Open your terminal and navigate to the directory where it was downloaded. For example, if it is in your Downloads folder

cd ~/Downloads
  1. Install the downloaded package by running the following, replacing mysql-apt-config_w.x.y-z.deb with the name of the file
sudo dpkg -i mysql-apt-config_w.x.y-z.deb
  1. You will see a few configurations. At the time of writing, they are:

MySQL Server & Cluster (Currently selected: mysql-8.0)

MySQL Tools & Connectors (Currently selected: Enabled)

MySQL Preview Packages (Currently selected: Disabled)

Leave everything as default and select Ok.

  1. Update the package information from the APT repository
sudo apt-get update
  1. Install the server
sudo apt-get install mysql-server

  1. Set a password for the root user

  2. Choose 'Strong Password Encryption'

  3. MySQL server is now running

    note

    You can start the server by running systemctl start mysql and stop the server by running systemctl stop mysql.

Generate required MySQL tables and stored procedures#

  1. Clone this repo https://github.com/bcit-pollination/web_server_pollination.git if you haven't already
git clone https://github.com/bcit-pollination/web_server_pollination.git
  1. Connect to the MySQL server (type in your root password after)
mysql -u root -p
  1. Setup the database tables (replace PATH with the location where you cloned the repo)
source /PATH/web_server_pollination/sql-scripts/SetUpDatabase.sql
  1. Setup the stored procedures (replace PATH with the location where you cloned the repo)
source /PATH/web_server_pollination/sql-scripts/CreateStoredProcedures.sql
  1. Disconnect from the MySQL server
exit

Set up MySQL database user#

  1. Connect to the MySQL server (type in your root password after)
mysql -u root -p
  1. Create a new MySQL account user for the server. Replace password with a password of your choice.
CREATE USER 'server'@'localhost' IDENTIFIED BY 'password';
  1. Grant the appropriate privileges
GRANT EXECUTE ON voting_system.* TO 'server'@'localhost';
  1. Disconnect from the MySQL server
exit
